Makbule sitesi — hızlı kurulum
==============================
1) Bu zip'i açın.
2) GitHub'da yeni bir repo oluşturun (public).
3) index.html ve (varsa) makbule.jpg dosyalarını repo köküne yükleyin.
4) Settings → Pages → Deploy from a branch → main / root → Save.
5) Yayın adresini NFC etiketine yazın.
